CHANCHO is a 4chan image downloader
that keeps watching threads for new changes

usage: chancho.py [-h] [-u URL [URL ...]] [-w WAIT] [-p [PRUNE]]

optional arguments:
  -h, --help            show this help message and exit
  -u URL [URL ...], --url URL [URL ...]
                        4chan threads to be added to the download list
  -w WAIT, --wait WAIT  seconds to wait between full downloads, 60 seconds
                        default
  -p [PRUNE], --prune [PRUNE]
                        archive old threads, 3600 seconds default (1h)
